Ruby Enumerable Cheat Sheet

List of Methods

Description
#each each {|x| ...}

Yields all elements.
#include?(value)

Returns true if any elements == value
#map or collect map/collect {|x| ...}

Returns an array with the results from block.
#each_with_index each_with_index {|x, i| ...}

Yields all elements with their indeces.
#sort sort [{|a, b| ...}]

Returns array sorted by <=> operator or by block. Default block { |a, b| a <=> b }.
#all? [a, b, c].all? {|letter| letter.length >= 3} #true

Passes each element of the collection to the given block. The method returns true if the block never returns false.
#count ar1 = [1,2,4,3] ar1.count # = 4

Returns the number of items in enum through enumeration.
